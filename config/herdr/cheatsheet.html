herdr quick keys

herdr 0.6.8 · stock theme · config: ~/.config/herdr/config.toml · effective bindings (your overrides + shipped defaults)
Prefix is ctrl + a — same as your old tmux prefix. Every ⌐-marked key below means: tap ctrl+a first, release, then the key.

Panes

⌐ %Split side-by-side (vertical)tmuxcustom
⌐ "Split stacked (horizontal)tmuxcustom
⌐ ←↓↑→Focus pane left / down / up / rightcustom (h/j/k/l still work)
⌐ tabCycle to next pane
⌐ shift+tabCycle to previous pane
⌐ fZoom / fullscreen togglecustom (default was z)
⌐ xClose pane
⌐ rResize mode (then h/j/k/l or arrows; esc to exit)
⌐ shift+pRename pane
⌐ eEdit scrollback in $EDITOR
Bare h/j/k/l are deliberately not bound directly (they'd be swallowed from the shell) — pane focus always goes through the prefix.

Tabs

⌐ cNew tab (prompts for a name)
⌐ shift+← / shift+→Previous / next tabcustom (replaces p/n)
⌐ 1…9Jump to tab 1–9
⌐ shift+tRename tab
⌐ shift+xClose tab

Workspaces & worktrees

⌐ shift+↑ / shift+↓Previous / next workspacecustom
⌐ wWorkspace picker
⌐ shift+nNew workspace
⌐ shift+wRename workspace
⌐ shift+dClose workspace (asks to confirm)
⌐ shift+gNew git worktree workspace
⌐ bToggle sidebar

Session & app

⌐ ?Help — live list of all bindings
⌐ gGo-to navigator — jump anywhere by name
⌐ sSettings
⌐ qDetach client (server + panes keep running)
⌐ shift+rReload config.toml
⌐ oOpen notification target — jump to the pane that pinged
Mouse: click to focus, wheel scrolls 3 lines/notch. Toasts are delivered in-app (ui.toast = "herdr"), agent labels show on pane borders.

CLI essentials

herdr                     # attach / launch session
herdr status              # client + server health
herdr server reload-config
herdr server stop
herdr config reset-keys   # back up + clear key overrides
herdr update              # stay current (stable channel)
Logs: ~/.config/herdr/herdr-{client,server}.log. Socket API auto-starts the server.

Where you differ from stock herdr

prefixctrl+a — stock is ctrl+b (kept from tmux)
split% / " — stock is v / minus
zoomprefix+f — stock is prefix+z
panesprefix+arrows — stock is prefix+h/j/k/l only
ws / tabsprefix+shift+↑↓ workspaces, prefix+shift+←→ tabs — stock has no workspace cycling; tabs were p/n
Theme and everything else on this sheet is a shipped default — if a key doesn't fire, ⌐ ? shows what your running server actually has loaded.
